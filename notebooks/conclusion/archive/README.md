# Archivo historico de notebooks de conclusion

Esta carpeta conserva notebooks historicos o backups del cierre del proyecto.

Estos notebooks pueden contener resultados intermedios, evaluaciones anteriores o analisis realizados antes del cierre definitivo. Se preservan por trazabilidad, pero no deben usarse como fuente vigente de metricas finales ni como evidencia principal del modelo seleccionado.

La fuente vigente de resultados finales esta en:

- `conclusion/final_metric_summary.csv`
- `conclusion/final_model_summary.csv`
- `conclusion/final_rul_bin_metrics.csv`
- `predictions/final_executable/all_final_metrics.csv`
- `predict_final.py`
- `conclusion/build_conclusion_artifacts.py`
- `conclusion/build_physical_operational_artifacts.py`
- `conclusion/build_physical_area_summary.py`
